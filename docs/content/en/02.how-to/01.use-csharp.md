## Install

```bash
dotnet add package JsonEvalRs
```

Targets: `netstandard2.0`, `netstandard2.1`, `net6.0`, `net8.0`, and `net10.0`.

## Evaluate and validate

```csharp
using JsonEvalRs;
using Newtonsoft.Json.Linq;

var schema = @"{
  'type': 'object',
  'properties': {
    'email': {
      'type': 'string',
      'rules': { 'required': { 'value': true, 'message': 'Email is required' } }
    }
  }
}".Replace("'", "\"");
var data = @"{ 'email': '' }".Replace("'", "\"");

using var evaluator = new JSONEval(schema);
evaluator.Evaluate(data);

ValidationResult validation = evaluator.Validate(data);
foreach (var error in validation.Errors) {
    Console.WriteLine($"{error.Path}: {error.Message}");
}

JObject compact = evaluator.GetEvaluatedSchema();
```

Dispose every directly owned `JSONEval`. `using` guarantees native handle cleanup.

## Update part of form

```csharp
var changedPaths = new[] { "user.email" };
evaluator.Evaluate(newData, paths: changedPaths);

JArray patches = evaluator.EvaluateDependents(
    changedPaths,
    data: newData,
    reEvaluate: true,
    includeSubforms: true);
```

`EvaluateDependents` defaults `reEvaluate` and `includeSubforms` to `true`. Set either false only when caller intentionally controls follow-up evaluation.

For validation only at selected fields:

```csharp
var result = evaluator.ValidatePaths(
    newData,
    paths: new List<string> { "user.email" });
```

## Get compact or resolved output

```csharp
JObject compact = evaluator.GetEvaluatedSchema();
JObject compactWithoutParams = evaluator.GetEvaluatedSchemaWithoutParams();
JObject resolved = evaluator.GetEvaluatedSchemaResolved();

byte[] compactMsgpack = evaluator.GetEvaluatedSchemaMsgpack();
byte[] resolvedMsgpack = evaluator.GetEvaluatedSchemaResolvedMsgpack();
```

- Compact output keeps `$params` and `$layout` references.
- `GetEvaluatedSchemaResolved()` starts without `$params` and merges resolved layout overlays.
- Getters take no `skipLayout` argument. Use distinct compact/resolved methods.
- `GetEvaluatedSchemaByPath(path)` and `GetEvaluatedSchemaByPaths(paths)` read compact output.

## Reload and cache schemas

```csharp
evaluator.ReloadSchema(nextSchema, data: nextData);
evaluator.ReloadSchemaMsgpack(schemaBytes, data: nextData);
evaluator.ReloadSchemaFromCache("pricing-v2", data: nextData);

using var cached = JSONEval.FromCache("pricing-v2", data: nextData);
```

`FromCache` requires schema inserted into parsed-schema cache first. Use cache APIs exposed by installed binding/application setup; do not assume constructing one evaluator registers a cache key.

## Subforms, time zones, and standalone logic

```csharp
evaluator.SetTimezoneOffset(420); // UTC+7

evaluator.EvaluateSubform("#/properties/items", itemData);
JArray patches = evaluator.EvaluateDependentsSubform(
    "#/properties/items",
    "quantity",
    itemData,
    reEvaluate: true,
    includeSubforms: true);

ulong logicId = evaluator.CompileLogic("{\"*\":[{\"var\":\"qty\"},2]}");
JToken value = evaluator.RunLogic(logicId, "{\"qty\":5}");
```

Do not share one mutable evaluator between concurrent operations. For request concurrency, create and dispose an evaluator per request, preferably from a parsed-schema cache.
