We'll cover how to install JSONEval-Rs and use it in Rust, C#, TypeScript (Web), and React Native. By the end of this tutorial, you'll be able to quickly evaluate JSON expressions against your data, leveraging the library's high-performance compilation and validation.

## Prerequisites

Before you start, make sure you have the required environment mapped to your language of choice:
- **Rust**: Latest stable Rust and Cargo.
- **C#**: .NET SDK 6.0+.
- **Web / TypeScript**: Node.js 18+ and a package manager (npm or yarn).
- **React Native**: React Native CLI or Expo initialized project.

---

## 1. Installation

Pick the installation method that matches your project platform.

### Rust
Add the dependency to your `Cargo.toml`:
```toml
[dependencies]
json-eval-rs = "0.0.69" # Or latest version
```

### C# / .NET
Install the NuGet package using the .NET CLI:
```bash
dotnet add package JsonEvalRs
```

### Web / TypeScript
Install the core API and a backend package of your choice (bundler or vanilla framework):
```bash
# For projects using Vite/Webpack
yarn add @json-eval-rs/webcore @json-eval-rs/bundler

# For Vanilla JS browser projects
yarn add @json-eval-rs/webcore @json-eval-rs/vanilla
```

> **Note**: For NodeJS use cases, install `@json-eval-rs/node` instead of the web backend packages.

### React Native
Install the custom React Native bindings:
```bash
yarn add @json-eval-rs/react-native
```

## 2. Your First Evaluation

Let's build a simple evaluation. We will use a rule (schema) that checks if an age is over 18, and data providing an age of 25.

### In Rust

```rust
use json_eval_rs::JSONEval;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // 1. Define your schema using JSON Schema format with logic blocks
    let schema = r#"{
        "type": "object",
        "properties": {
            "age": {
                "rules": {
                    "minValue": { "value": 18, "message": "Must be 18 or older" }
                }
            }
        }
    }"#;

    // 2. Initialize the evaluator tool (compiles the schema logic)
    let mut eval = JSONEval::new(schema, None, None)?;
    
    // 3. Define the data source
    let data = r#"{"age": 25}"#;

    // 4. Evaluate the data
    eval.evaluate(data, None, None)?;

    // 5. Validate rules and check for errors
    let validation = eval.validate(data, None, None)?;
    if !validation.has_error {
        println!("✅ Data is valid!");
    } else {
        println!("❌ Validation errors: {:?}", validation.errors);
    }

    Ok(())
}
```

### In C#

```csharp
using JsonEvalRs;
using System;

class Program {
    static void Main() {
        var schema = @"{
            ""type"": ""object"",
            ""properties"": {
                ""age"": {
                    ""rules"": {
                        ""minValue"": { ""value"": 18, ""message"": ""Must be 18 or older"" }
                    }
                }
            }
        }";

        // Wrap JSONEval within a using block to correctly discard memory resources
        using (var eval = new JSONEval(schema)) {
            var data = @"{""age"": 25}";
            
            // Execute the evaluation
            eval.Evaluate(data);
            
            // Get validation statuses
            var validation = eval.Validate(data);

            if (!validation.HasError) {
                Console.WriteLine("✅ Data is valid!");
            }
        }
    }
}
```

### In Web (TypeScript)

When using a bundler on the Web, you should supply the WebAssembly module on initialization. (If using the `@json-eval-rs/vanilla` package, import from that module instead).

```typescript
import { JSONEval } from '@json-eval-rs/webcore';
import * as wasmModule from '@json-eval-rs/bundler'; // or '@json-eval-rs/vanilla'

async function run() {
    const schema = {
      type: "object",
      properties: {
        age: {
          rules: {
            minValue: { value: 18, message: "Must be 18 or older" }
          }
        }
      }
    };

    // 1. Initialize logic evaluator
    const evalInstance = new JSONEval({
      schema: JSON.stringify(schema),
      wasmModule
    });

    const data = { age: 25 };

    try {
      // 2. Run validations across logic statements
      const validation = await evalInstance.validate({ 
        data: JSON.stringify(data) 
      });

      if (!validation.has_error) {
          console.log("✅ Data is valid!");
      }
    } finally {
      // 3. Make sure to free memory when done!
      evalInstance.free();
    }
}

run();
```
