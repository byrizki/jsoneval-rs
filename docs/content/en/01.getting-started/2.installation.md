Install public binding matching runtime. Do not install `@json-eval-rs/webcore` directly; it is implementation dependency of public web packages.

## Rust

```toml
[dependencies]
json-eval-rs = "0.0.108"
```

## C# / .NET

```bash
dotnet add package JsonEvalRs
```

Package targets .NET Standard 2.0/2.1 and .NET 6, 8, and 10.

## Web and TypeScript

```bash
# Vite, Webpack, Rollup, or Next.js bundling
npm install @json-eval-rs/bundler

# Node.js or server-side rendering
npm install @json-eval-rs/node

# Browser code that explicitly initializes WASM
npm install @json-eval-rs/vanilla
```

Use exactly one runtime package per execution target. `@json-eval-rs/vanilla` requires `JSONEval.initWasm()` before use.

## React Native

```bash
npm install @json-eval-rs/react-native
```

Then follow React Native native-module setup for iOS and Android in its package README.
