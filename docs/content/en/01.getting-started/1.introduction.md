## Key features

- 🚀 **High performance**: Custom JSON Logic compiler with zero-copy caching
- 🔄 **Topological sorting**: Dependency-aware processing to ensure correct execution order
- 📊 **Schema validation**: Built-in validation with detailed error mapping
- 🌐 **Multi-platform**: Native bindings for Rust, C#/.NET, Web (WASM), and React Native
- 💾 **Smart caching**: Content-based caching with Arc-based zero-copy storage
- 🔍 **Dependency tracking**: Detects field dependencies for targeted re-evaluations
- 📐 **SIMD optimized**: Uses `simd-json` to parse JSON formats quickly
- ⚡ **Selective evaluation**: Computes only specified fields to save CPU cycles
- 🌍 **Timezone support**: Configurable timezone offsets for date calculations

Below we cover how to install and use JSONEval-Rs. By the end, you'll know how to evaluate JSON expressions quickly using the library's compiler and validator.

## Prerequisites

Make sure you have the required environment setup for your language:
- **Rust**: Latest stable Rust and Cargo.
- **C#**: .NET SDK 6.0+.
- **Web / TypeScript**: Node.js 18+ and a package manager (npm or yarn).
- **React Native**: React Native CLI or Expo initialized project.
