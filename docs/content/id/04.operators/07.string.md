## Gambaran Umum

Operator teks menyediakan kemampuan manipulasi teks yang komprehensif untuk pemformatan, analisis, pencarian, dan transformasi data string.

## `cat` - Menggabungkan (Concatenate)

Menggabungkan beberapa nilai menjadi satu string.

### Contoh
```json
{"cat": ["Halo", " ", "Dunia"]}  // → "Halo Dunia"
{"cat": ["Total: ", 42]}         // → "Total: 42"
```

---

## `substr` - Substring

Mengambil bagian tertentu dari sebuah string.

### Contoh
```json
{"substr": ["Halo Dunia", 0, 4]}   // → "Halo"
{"substr": ["Halo Dunia", 5]}      // → "Dunia"
```

---

## `search` / `SEARCH` - Mencari Teks

Mencari posisi teks di dalam string (tidak peka huruf besar/kecil). Mengembalikan indeks berbasis 1.

### Contoh
```json
{"search": ["Dunia", "Halo Dunia"]}     // → 6
{"search": ["xyz", "Halo Dunia"]}       // → null
```

---

## `left` / `right` / `mid` - Ekstraksi Posisi

Mengambil karakter dari sisi kiri, kanan, atau tengah string.

### Contoh
```json
{"left": ["Halo Dunia", 4]}        // → "Halo"
{"right": ["Halo Dunia", 5]}       // → "Dunia"
{"MID": ["Halo Dunia", 6, 5]}      // → "Dunia"
```

---

## `len` / `length` - Panjang Teks

Mengembalikan jumlah karakter dalam string atau jumlah elemen dalam array/objek.

### Contoh
```json
{"len": "Halo"}                    // → 4
```

---

## `splittext` / `splitvalue` - Memecah Teks

Memecah string berdasarkan pemisah (separator) tertentu.

### Contoh
```json
// Ambil elemen ke-2 (indeks 1)
{"splittext": ["A,B,C", ",", 1]}   // → "B"

// Ubah jadi array
{"splitvalue": ["A,B,C", ","]}     // → ["A", "B", "C"]
```

---

## `stringformat` - Format Angka ke Teks

Memformat angka menjadi string dengan pemisah ribuan, desimal, awalan, dan akhiran.

### Contoh
```json
// Format mata uang
{"STRINGFORMAT": [1234.5, 2, "$"]}   // → "$1,234.50"

// Format persentase
{"stringformat": [75, 0, "", "%"]}   // → "75%"
```

## Contoh Kompleks

### Membuat Nama Lengkap
```json
{"cat": [
  {"var": "namaDepan"},
  " ",
  {"var": "namaBelakang"}
]}
```

### Validasi Format Email Sederhana
```json
{"and": [
  {">": [{"search": ["@", {"var": "email"}]}, 0]},
  {">": [{"search": [".", {"var": "email"}]}, {"search": ["@", {"var": "email"}]}]}
]}
```

---

## Praktik Terbaik

1. **Gunakan `cat` untuk membangun pesan** yang melibatkan variabel agar lebih mudah dibaca.
2. **Selalu berikan nilai default** menggunakan `ifnull` sebelum menggabungkan variabel yang mungkin bernilai null.
3. **Gunakan `stringformat` untuk tampilan angka** di antarmuka pengguna (UI) agar terlihat rapi (misal: mata uang).
4. **Indeks pencarian (`search`) berbasis 1**, berbeda dengan `substr` yang berbasis 0. Perhatikan ini saat melakukan perhitungan posisi.
