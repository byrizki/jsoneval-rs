## Gambaran Umum

Fungsi tanggal menyediakan kemampuan komprehensif untuk bekerja dengan tanggal, waktu, dan interval. Operator ini menangani analisis tanggal, pemformatan, aritmatika, dan perbandingan.

### Standar Format Tanggal

JSON-Eval-RS menggunakan format **ISO 8601**:
- **Hanya Tanggal**: `"2024-01-15"`
- **Datetime**: `"2024-01-15T14:30:00Z"`

### Kesadaran Zona Waktu

Semua operator tanggal/waktu menghormati aturan zona waktu yang dikonfigurasi. Secara default adalah UTC, namun dapat disesuaikan melalui `set_timezone_offset(minutes)`.

## `today` / `TODAY` - Tanggal Hari Ini

Mengembalikan tanggal saat ini pada waktu tengah malam (00:00:00).

### Contoh
```json
{"today": null}  // → "2024-01-15T00:00:00.000Z"
```

---

## `now` / `NOW` - Waktu Sekarang

Mengembalikan tanggal dan waktu saat ini dengan presisi penuh.

### Contoh
```json
{"now": null}  // → "2024-01-15T14:30:45.123Z"
```

---

## `year` / `month` / `day` - Ekstraksi Komponen

Mengambil komponen tahun, bulan (1-12), atau hari (1-31) dari sebuah string tanggal.

### Contoh
```json
{"year": "2024-06-15"}   // → 2024
{"month": "2024-06-15"}  // → 6
{"day": "2024-06-15"}    // → 15
```

---

## `date` / `DATE` - Membuat Tanggal

Membuat objek tanggal dari komponen tahun, bulan, dan hari.

### Contoh
```json
{"date": [2024, 12, 25]}  // → "2024-12-25T00:00:00.000Z"
```

---

## `days` / `DAYS` - Selisih Hari

Menghitung jumlah hari di antara dua tanggal.

### Contoh
```json
{"days": ["2024-01-20", "2024-01-15"]}  // → 5
```

---

## `dateformat` / `DATEFORMAT` - Format Tanggal

Memformat string tanggal menggunakan format yang telah ditentukan atau kustom.

### Format yang Telah Ditentukan
- **`"iso"`**: `"2024-01-15"`
- **`"short"`**: `"01/15/2024"` (Format US)
- **`"long"`**: `"January 15, 2024"`
- **`"eu"`**: `"15/01/2024"` (Format Eropa)

### Contoh
```json
{"dateformat": ["2024-01-15", "long"]}    // → "January 15, 2024"
{"dateformat": ["2024-01-15", "%d-%m-%Y"]} // → "15-01-2024"
```

---

## `yearfrac` / `YEARFRAC` - Fraksi Tahun

Menghitung pecahan tahun di antara dua tanggal (berguna untuk perhitungan bunga atau umur desimal).

### Contoh
```json
{"YEARFRAC": ["1990-03-15", "2023-07-10"]}  // → ~33.32
```

---

## `datedif` / `DATEDIF` - Perbedaan Tanggal

Menghitung perbedaan antara dua tanggal dalam satuan tertentu (Y, M, D, YM, YD, MD).

### Contoh
```json
{"DATEDIF": ["1990-03-15", "2023-07-10", "Y"]}  // → 33 (Tahun penuh)
```

---

## Praktik Terbaik

1. **Selalu gunakan format ISO 8601** (`YYYY-MM-DD`) untuk konsistensi dan menghindari ambiguitas.
2. **Gunakan `DATEDIF` dengan unit "Y"** untuk perhitungan umur yang akurat.
3. **Berikan penanganan nilai null** sebelum melakukan operasi tanggal untuk menghindari error.
4. **Gunakan `dateformat` hanya pada tahap akhir** (tampilan ke pengguna), tetap gunakan format ISO untuk logika internal.
