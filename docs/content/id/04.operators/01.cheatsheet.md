## "Sebutkan Apa yang Ingin Kamu Lakukan..."

### Akses Variable dan Parameter

| Tugas | Operator | Contoh |
|------|----------|---------|
| Mengakses variabel | `var` | `{"var": "user.name"}` |
| Memberikan default variabel | `var` dengan default | `{"var": ["age", 0]}` |
| Cek kolom yang hilang | `missing` | `{"missing": ["email", "phone"]}` |
| Mengembalikan nilai mentah | `return` | `{"return": "nilai literal"}` |

### Membandingkan Nilai

| Tugas | Operator | Contoh |
|------|----------|---------|
| Cek kesamaan (longgar) | `==` | `{"==": [a, b]}` |
| Cek kesamaan (ketat) | `===` | `{"===": [a, b]}` |
| Kurang dari | `<` | `{"<": [umur, 18]}` |
| Lebih dari | `>` | `{">": [skor, 100]}` |
| Cek jangkauan (range) | `and` + perbandingan | `{"and": [{">=": [val, min]}, {"<=": [val, max]}]}` |

### Logika dan Kondisional

| Tugas | Operator | Contoh |
|------|----------|---------|
| Kondisional (jika-maka-lainnya) | `if` | `{"if": [kondisi, maka, lainnya]}` |
| Beberapa kondisi (DAN) | `and` | `{"and": [kond1, kond2]}` |
| Kondisi alternatif (ATAU) | `or` | `{"or": [val1, val2]}` |
| Negasi kondisi | `not` atau `!` | `{"!": kondisi}` |
| Nilai default jika null | `ifnull` | `{"ifnull": [nilai, "default"]}` |

### Operasi Matematika

| Tugas | Operator | Contoh |
|------|----------|---------|
| Menjumlahkan angka | `+` | `{"+": [1, 2, 3]}` |
| Mengurangi | `-` | `{"-": [10, 3]}` |
| Mengali | `*` | `{"*": [harga, jumlah]}` |
| Membagi | `/` | `{"/": [total, hitung]}` |
| Pangkat | `^` atau `pow` | `{"pow": [2, 3]}` |
| Nilai absolut | `abs` | `{"abs": -5}` |
| Nilai maksimum | `max` | `{"max": [1, 5, 3]}` |
| Nilai minimum | `min` | `{"min": [1, 5, 3]}` |
| Bulatkan desimal | `round` | `{"round": [3.14159, 2]}` |
| Bulatkan ke atas | `roundup` | `{"roundup": [3.1]}` |
| Bulatkan ke bawah | `rounddown` | `{"rounddown": [3.9]}` |

### String

| Tugas | Operator | Contoh |
|------|----------|---------|
| Menggabungkan string | `cat` | `{"cat": ["Halo", " ", "Dunia"]}` |
| Ambil sebagian teks | `substr` | `{"substr": [teks, 0, 5]}` |
| cari teks di dalam string | `search` | `{"search": ["cari", "teks ini"]}` |
| Panjang string | `len` atau `length` | `{"len": teks}` |
| Memecah teks ke array | `splitvalue` | `{"splitvalue": ["a,b,c", ","]}` |
| Format angka ke string | `stringformat` | `{"stringformat": [1234.5, 2, "$"]}` |

### Date

| Tugas | Operator | Contoh |
|------|----------|---------|
| Ambil tanggal hari ini | `today` | `{"today": null}` |
| Ambil tahun dari tanggal | `year` | `{"year": tanggal}` |
| Selisih hari antar tanggal | `days` | `{"days": [akhir, awal]}` |
| Menghitung umur | `DATEDIF` dengan "Y" | `{"DATEDIF": [tgl_lahir, hari_ini, "Y"]}` |

### Array

| Tugas | Operator | Contoh |
|------|----------|---------|
| Transformasi elemen | `map` | `{"map": [array, logika]}` |
| Saring array dengan kondisi | `filter` | `{"filter": [array, kondisi]}` |
| Cek jika semua cocok | `all` | `{"all": [array, kondisi]}` |
| Cek jika ada yang cocok | `some` | `{"some": [array, kondisi]}` |
| Cek nilai di dalam array | `in` | `{"in": [nilai, array]}` |
| Jumlahkan nilai array | `sum` | `{"sum": [array]}` |
| Panjang array | `length` | `{"length": array}` |

### Table & Lookup

| Tugas | Operator | Contoh |
|------|----------|---------|
| Ambil nilai dari tabel | `VALUEAT` | `{"VALUEAT": [tabel, idx, "kolom"]}` |
| Cari indeks baris | `INDEXAT` | `{"INDEXAT": [val, tabel, "kolom"]}` |
| Cocokkan dan ambil indeks | `MATCH` | `{"MATCH": [tabel, val, "kolom"]}` |
| Pilih kecocokan pertama | `CHOOSE` | `{"CHOOSE": [tabel, val, "kolom"]}` |

---

**Tips:** Jika ragu, selalu periksa dokumentasi operator spesifik untuk contoh yang lebih lengkap!
