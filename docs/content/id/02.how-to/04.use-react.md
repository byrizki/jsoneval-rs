## Instal

```bash
yarn add @json-eval-rs/bundler
```

Gunakan wrapper bundler publik. Jangan import `@json-eval-rs/webcore` internal atau mengirim opsi constructor `wasmModule`.

## Buat hook evaluator

```tsx
import { useEffect, useState } from 'react';
import type { JSONEval as JSONEvalType } from '@json-eval-rs/bundler';

export function useJSONEval(schema: object) {
  const [evaluator, setEvaluator] = useState<JSONEvalType | null>(null);
  const [error, setError] = useState<Error | null>(null);

  useEffect(() => {
    let instance: JSONEvalType | null = null;
    let active = true;

    async function load() {
      try {
        const { JSONEval } = await import('@json-eval-rs/bundler');
        instance = new JSONEval({ schema });
        await instance.init();
        if (active) setEvaluator(instance);
      } catch (cause) {
        if (active) setError(cause instanceof Error ? cause : new Error(String(cause)));
      }
    }

    load();
    return () => {
      active = false;
      instance?.free();
    };
  }, [schema]);

  return { evaluator, error };
}
```

Pertahankan referensi `schema` stabil (`useMemo` atau konstanta module) agar evaluator tidak dibuat ulang setiap render.

## Validasi input form

```tsx
const { evaluator, error } = useJSONEval(schema);
const [validation, setValidation] = useState<any>(null);

async function validate(age: string) {
  if (!evaluator) return;
  setValidation(await evaluator.validate({
    data: { age: Number(age) || 0 }
  }));
}
```

Gunakan `evaluate({ data, paths })` untuk schema terhitung; gunakan `validatePaths({ data, paths })` bila hanya memvalidasi field tertentu.

## Cleanup

`JSONEval` memiliki resource WASM. Panggil `free()` sekali untuk tiap instance, biasanya pada cleanup `useEffect`. Jangan free saat evaluasi masih memakai instance.

Untuk pemilihan package dan output compact/resolved, lihat [Web & NodeJS](use-web-node).
