## Fitur utama

- 🚀 **Performa tinggi**: Menggunakan compiler JSON Logic dengan memori management zero-copy caching.
- 🔄 **Evaluasi paralel**: Memroses node secara otomatis berdasarkan ketergantungan (Topological sorting).
- 📊 **Validasi schema**: Me-return mapping error log bawaan langsung dari Rust.
- 🌐 **Multi-platform**: Bisa digunakan langsung dari Rust, C#/.NET, Web (WASM), hingga React Native.
- 💾 **Caching pintar**: Menyimpan log kalkulasi via referensi Arc agar tidak membebani memori.
- 🔍 **Pelacakan ketergantungan**: Mendeteksi letak field di form yang rentan terpengaruh update terbaru secara otomatis.
- 📐 **Optimasi SIMD**: Memakai package `simd-json` native agar parse string C++ ke JSON sangat enteng.
- ⚡ **Evaluasi terfokus**: Cuma menghitung ulang blok spesifik yang diklik untuk memangkas resourse CPU.
- 🌍 **Dukungan timezone**: Hitung mundur dan kalkulasi tanggal berdasarkan offset regional lokal pengguna.

---

Panduan ini membahas instalasi dan cara penggunaan JSONEval-Rs di platform Anda. Kita akan langsung membuat evaluasi rule tanpa konfigurasi berbelit.

## Prasyarat perangkat

Pastikan perangkat Anda sudah menginstal compiler atau SDK berikut:
- **Rust**: Rust CLI dan Cargo vStable terbaru.
- **C#**: .NET SDK 6.0 ke atas.
- **Web / TypeScript**: Node.js v18 dan NodeJS package manager (npm/yarn/pnpm).
- **React Native**: Repo yang di-init dari React Native CLI atau Expo.
