`json-eval-rs` mengevaluasi JSON Logic di dalam JSON schema. Pustaka ini juga memvalidasi data, melacak ketergantungan field, me-resolve layout overlay, dan mendukung evaluasi ulang selektif.

## Kemampuan

- **Evaluasi schema** — mengevaluasi value, rule, condition, option, table, dan data layout.
- **Validasi** — mengembalikan error validasi terstruktur untuk rule schema.
- **Pembaruan dependensi** — mengevaluasi ulang field yang terpengaruh path perubahan.
- **Parsing yang dapat digunakan ulang** — parse sekali memakai `ParsedSchema`, lalu buat evaluator untuk banyak data.
- **Bentuk hasil schema** — ambil hasil ringkas, tanpa `$params`, dengan layout terselesaikan, atau MessagePack.
- **Binding** — Rust, C#/.NET, WebAssembly browser/Node, dan React Native.
- **Runtime logic** — evaluasi logic mandiri atau compile untuk eksekusi berulang.

## Runtime yang didukung

| Runtime | Paket atau crate |
| --- | --- |
| Rust | `json-eval-rs` |
| C#/.NET | `JsonEvalRs` |
| Web app dengan bundler | `@json-eval-rs/bundler` |
| Browser dengan script tag | `@json-eval-rs/vanilla` |
| Node.js dan SSR | `@json-eval-rs/node` |
| React Native | `@json-eval-rs/react-native` |
