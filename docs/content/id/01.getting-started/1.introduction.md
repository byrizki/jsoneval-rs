## Fitur Utama

- 🚀 **Performa Tinggi**: Menggunakan compiler JSON Logic khusus dengan pre-compilation dan caching yang sangat cepat.
- 🔄 **Evaluasi Paralel**: Proses multi-thread yang sanggup mengurutkan tugas secara otomatis berdasarkan ketergantungannya.
- 📊 **Validasi Schema**: Sudah dilengkapi validasi dan laporan error yang detail.
- 🌐 **Multi-Platform**: Bisa digunakan langsung untuk Rust, C#/.NET, Web (WASM), dan React Native.
- 💾 **Caching Pintar**: Penyimpanan data berbasis konten yang tidak membebani memori (zero-copy).
- 🔍 **Pelacakan Ketergantungan**: Mendeteksi otomatis kolom mana saja yang saling bergantung agar evaluasi jadi lebih efisien.
- 📐 **Optimasi SIMD**: Memakai `simd-json` untuk pembacaan JSON yang super cepat.
- ⚡ **Evaluasi Selektif**: Hanya mengevaluasi kolom tertentu saja untuk meningkatkan kecepatan.
- 🌍 **Dukungan Timezone**: Pengaturan perbedaan waktu (offset) yang fleksibel untuk operasi tanggal/waktu.

---

Di sini kita akan mempelajari cara instalasi JSONEval-Rs dan cara menggunakannya di Rust, C#, TypeScript (Web), dan React Native. Setelah selesai mengikuti panduan ini, kamu akan bisa mengevaluasi ekspresi JSON dengan cepat menggunakan fitur kompilasi dan validasi yang tersedia di pustaka ini.

## Prasyarat

Sebelum mulai, pastikan kamu sudah menyiapkan lingkungan sesuai bahasa pemrograman yang kamu pilih:
- **Rust**: Versi stable Rust dan Cargo terbaru.
- **C#**: .NET SDK 6.0 ke atas.
- **Web / TypeScript**: Node.js 18 ke atas dan package manager (npm atau yarn).
- **React Native**: Proyek yang dibuat dengan React Native CLI atau Expo.
