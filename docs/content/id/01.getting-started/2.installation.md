Pilih metode instalasi untuk platformmu.

### Rust
Tambahkan versi crate berikut ke file `Cargo.toml`:
```toml
[dependencies]
json-eval-rs = "0.0.88" # Atau versi terbaru
```

### C# / .NET
Instal paket via command-line NuGet:
```bash
dotnet add package JsonEvalRs
```

### Web / TypeScript
Sertakan package webcore beserta module WebAssembly target Anda:
```bash
# Bagi proyek modern framework semacam Vite/Webpack
yarn add @json-eval-rs/webcore @json-eval-rs/bundler

# Bagi skrip website JS native tanpa bundler (Vanilla JS)
yarn add @json-eval-rs/webcore @json-eval-rs/vanilla
```

> **Catatan**: Kalau Anda mengeksekusi script API di backend berbasis Node.js murni, pastikan menginstal `@json-eval-rs/node` alih-alih opsi di atas.

### React Native
Instal binding asli khusus untuk JSI bridge ponsel:
```bash
yarn add @json-eval-rs/react-native
```
