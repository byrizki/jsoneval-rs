Pilih metode instalasi yang sesuai dengan platform proyekmu.

### Rust
Tambahkan baris berikut ke file `Cargo.toml`:
```toml
[dependencies]
json-eval-rs = "0.0.86" # Atau versi terbaru
```

### C# / .NET
Instal paket NuGet menggunakan .NET CLI:
```bash
dotnet add package JsonEvalRs
```

### Web / TypeScript
Instal API core dan paket backend pilihanmu (bundler atau vanilla framework):
```bash
# Untuk proyek yang menggunakan Vite/Webpack
yarn add @json-eval-rs/webcore @json-eval-rs/bundler

# Untuk proyek browser JS biasa (Vanilla JS)
yarn add @json-eval-rs/webcore @json-eval-rs/vanilla
```

> **Catatan**: Untuk penggunaan di NodeJS, instal `@json-eval-rs/node` saja, bukan paket backend web.

### React Native
Instal binding khusus untuk React Native:
```bash
yarn add @json-eval-rs/react-native
```
