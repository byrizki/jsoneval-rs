Instal binding publik sesuai runtime. Jangan instal `@json-eval-rs/webcore` langsung; paket tersebut adalah dependensi implementasi paket web publik.

## Rust

```toml
[dependencies]
json-eval-rs = "0.0.107"
```

## C# / .NET

```bash
dotnet add package JsonEvalRs
```

Paket mendukung .NET Standard 2.0/2.1 serta .NET 6, 8, dan 10.

## Web dan TypeScript

```bash
# Vite, Webpack, Rollup, atau Next.js dengan bundler
npm install @json-eval-rs/bundler

# Node.js atau server-side rendering
npm install @json-eval-rs/node

# Kode browser yang menginisialisasi WASM secara eksplisit
npm install @json-eval-rs/vanilla
```

Gunakan satu paket runtime untuk tiap target eksekusi. `@json-eval-rs/vanilla` membutuhkan `JSONEval.initWasm()` sebelum dipakai.

## React Native

```bash
npm install @json-eval-rs/react-native
```

Lalu ikuti setup native module iOS dan Android pada README paket React Native.

Berikutnya: [jalankan evaluasi pertama](./3.quick-start.md).
