Mari kita coba membuat evaluasi sederhana. Kita akan membuat aturan (schema) untuk mengecek apakah umur seseorang di atas 18, menggunakan data dengan umur 25.

### Rust

```rust
use json_eval_rs::JSONEval;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // 1. Tentukan schema menggunakan format JSON Schema dengan blok logika
    let schema = r#"{
        "type": "object",
        "properties": {
            "age": {
                "rules": {
                    "minValue": { "value": 18, "message": "Harus berumur 18 atau lebih" }
                }
            }
        }
    }"#;

    // 2. Inisialisasi alat evaluator (menyusun logika schema)
    let mut eval = JSONEval::new(schema, None, None)?;
    
    // 3. Tentukan data yang akan dicek
    let data = r#"{"age": 25}"#;

    // 4. Evaluasi datanya
    eval.evaluate(data, None, None)?;

    // 5. Validasi aturan dan cek jika ada error
    let validation = eval.validate(data, None, None)?;
    if !validation.has_error {
        println!("✅ Data valid!");
    } else {
        println!("❌ Ada error validasi: {:?}", validation.errors);
    }

    Ok(())
}
```

### C#

```csharp
using JsonEvalRs;
using System;

class Program {
    static void Main() {
        var schema = @"{
            ""type"": ""object"",
            ""properties"": {
                ""age"": {
                    ""rules"": {
                        ""minValue"": { ""value"": 18, ""message"": ""Harus berumur 18 atau lebih"" }
                    }
                }
            }
        }";

        // Gunakan blok 'using' agar JSONEval menghapus penggunaan memori dengan benar saat selesai
        using (var eval = new JSONEval(schema)) {
            var data = @"{""age"": 25}";
            
            // Jalankan evaluasi
            eval.Evaluate(data);
            
            // Ambil status validasi
            var validation = eval.Validate(data);

            if (!validation.HasError) {
                Console.WriteLine("✅ Data valid!");
            }
        }
    }
}
```

### React Native

```typescript
import React, { useState } from 'react';
import { View, Text, TextInput } from 'react-native';
import { useJSONEval } from '@json-eval-rs/react-native';

const schema = {
  type: 'object',
  properties: {
    age: {
      rules: {
        minValue: { value: 18, message: 'Kamu harus berumur 18 atau lebih' }
      }
    }
  }
};

export default function AgeValidator() {
  const [age, setAge] = useState('');
  const [errorMsg, setErrorMsg] = useState('');
  
  // Inisialisasi evaluator
  const evalInstance = useJSONEval({ schema });

  const handleChange = async (text: string) => {
    setAge(text);

    if (evalInstance) {
      const data = { age: parseInt(text, 10) || 0 };
      const validation = await evalInstance.validate({ 
        data: JSON.stringify(data) 
      });
      
      setErrorMsg(validation.has_error ? validation.errors.age?.message : '');
    }
  };

  return (
    <View>
      <TextInput 
        value={age}
        onChangeText={handleChange}
        keyboardType="numeric"
        placeholder="Masukkan umurmu"
      />
      {errorMsg ? <Text style={{color: 'red'}}>{errorMsg}</Text> : null}
    </View>
  );
}
```

### Web (Node.js)

Untuk lingkungan backend JavaScript menggunakan Node.js, gunakan paket `@json-eval-rs/node`. Paket ini akan otomatis memuat library pendukungnya.

```typescript
import { JSONEval } from '@json-eval-rs/webcore';
import '@json-eval-rs/node';

async function run() {
    const schema = {
      type: "object",
      properties: {
        age: {
          rules: {
            minValue: { value: 18, message: "Harus berumur 18 atau lebih" }
          }
        }
      }
    };

    // 1. Inisialisasi logic evaluator
    const evalInstance = new JSONEval({
      schema: JSON.stringify(schema)
    });

    const data = { age: 25 };

    try {
      // 2. Jalankan validasi pada tiap logika
      const validation = await evalInstance.validate({ 
        data: JSON.stringify(data) 
      });

      if (!validation.has_error) {
          console.log("✅ Data valid!");
      }
    } finally {
      // 3. Pastikan untuk menghapus penggunaan memori jika sudah selesai!
      evalInstance.free();
    }
}

run();
```

### Web (Bundler)

Saat menggunakan bundler (Vite, Webpack, atau framework seperti Next.js/NuxtJS), kamu harus menyertakan modul WebAssembly saat inisialisasi:

```typescript
import { JSONEval } from '@json-eval-rs/webcore';
import * as wasmModule from '@json-eval-rs/bundler';

async function run() {
    // ... schema dan data yang sama ...
    const schema = { /* ... */ };
    const data = { age: 25 };

    const evalInstance = new JSONEval({
      schema: JSON.stringify(schema),
      wasmModule
    });

    try {
      const validation = await evalInstance.validate({ data: JSON.stringify(data) });
      if (!validation.has_error) console.log("✅ Data valid!");
    } finally {
      evalInstance.free();
    }
}

run();
```

### Web (Vanilla JS)

Untuk file HTML/JS biasa tanpa bundler, gunakan modul ES standar dari paket `@json-eval-rs/vanilla`:

```typescript
import { JSONEval } from '@json-eval-rs/webcore';
import * as wasmModule from '@json-eval-rs/vanilla';

async function run() {
    // Inisialisasi sama seperti cara pakai bundler:
    const evalInstance = new JSONEval({ schema: JSON.stringify(schema), wasmModule });
    // ...
}
```
